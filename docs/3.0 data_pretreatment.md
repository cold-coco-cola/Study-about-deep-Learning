`PandaFrame.mean()`会对所有值进行取平，加上`numeric_only=True`可以只取数值的平均。

同样地，`get_dummies()`默认为bull值，`dtype=int`可改变数据类型为int，改变数据类型为数值后，之后的操作就能正常进行不会报错。

```python
nan_numer = data.isnull().sum(axis=0)  #isnull()&isna() 判断是否为nan并返回布尔值，最后总体返回bull列表，再sum求True的数量
nan_max_id = nan_numer.idxmax()    #找到最大值的索引（index）
data = data.drop([nan_max_id], axis=1) #删除该列（）
data

inputs = data.iloc[:, 0:2]
inputs = inputs.fillna(inputs.mean(numeric_only=True))
print(inputs)
```

```text
   NumRooms   Price
0       3.0  127500
1       2.0  106000
2       4.0  178100
3       3.0  140000
```

`isna()`:判断是否缺失 `idxmax()`:最大值的索引 `axis=0`:对象为0维，在`sum()`中表示把0维（方向）压缩（压缩对象），在`data.drop()`中表示操作对象为列 熟悉这些代码的含义
